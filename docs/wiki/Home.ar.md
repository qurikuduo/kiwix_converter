# ويكي Kiwix Converter

مرحباً بك في ويكي Kiwix Converter.

## نظرة عامة

يقوم Kiwix Converter بتحويل أرشيفات ZIM الخاصة بـ kiwix-desktop إلى ملفات Markdown على مستوى المقالات وملفات JSON للبيانات الوصفية وملفات JSONL مجزأة مناسبة لأنظمة RAG.

## المجالات الرئيسية

- واجهة سطح المكتب: الإعدادات، الفحص، التحكم في المهام، السجل، وسجلات التشغيل
- محرك التحويل: التكامل مع `zimdump`، استخراج المحتوى الرئيسي، التحويل إلى Markdown، تصدير الصور، وإعادة كتابة الروابط
- الاستمرارية: حفظ الإعدادات ونقاط التوقف والبيانات الوصفية والسجلات في SQLite
- الأتمتة: البناء المستمر، التغليف، ونشر الإصدارات وفق semantic versioning
