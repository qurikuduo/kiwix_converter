# ويكي Kiwix Converter

مرحباً بك في ويكي Kiwix Converter.

## نظرة عامة

يقوم Kiwix Converter بتحويل أرشيفات ZIM الخاصة بـ kiwix-desktop إلى ملفات Markdown على مستوى المقالات وملفات JSON للبيانات الوصفية وملفات JSONL مجزأة مناسبة لأنظمة RAG.

## المجالات الرئيسية

- واجهة سطح المكتب: الإعدادات، الفحص، التحكم في المهام، السجل، وسجلات التشغيل
- محرك التحويل: التكامل مع `zimdump`، استخراج المحتوى الرئيسي، التحويل إلى Markdown، تصدير الصور، وإعادة كتابة الروابط
- الاستمرارية: حفظ الإعدادات ونقاط التوقف والبيانات الوصفية والسجلات في SQLite
- الأتمتة: البناء المستمر، التغليف، ونشر الإصدارات وفق semantic versioning

## البنية وتدفق البيانات

- يتولى `KiwixConverter.WinForms` غلاف التشغيل للمستخدم، بينما يتولى `KiwixConverter.Core` الفحص والتحويل والمزامنة والاستمرارية.
- يقوم فحص المجلد بعمل upsert لقائمة ZIM داخل SQLite قبل بدء التحويل.
- يستخدم التحويل `zimdump` لجلب البيانات الوصفية و HTML ثم ينتج Markdown وملفات JSON الوصفية وملفات chunks ومجلدات الصور.
- يمكن لمزامنة WeKnora تحميل معرّفات النماذج من `/api/v1/models`، وإنشاء أو إعادة استخدام KB بإعدادات chunk، ورفع Markdown لكل مقالة مع حالة قابلة للاستئناف.

## بدء سريع

بالنسبة للمستخدم غير التقني، المسار الأسهل هو:

1. تنزيل أحدث Windows release zip.
2. تثبيت .NET 8 Desktop Runtime.
3. تثبيت `zimdump` وإضافته إلى `PATH` أو اختيار `zimdump.exe` من داخل التطبيق.
4. تشغيل التطبيق، وتحديد مجلد `kiwix-desktop` ومجلد الإخراج، ثم فحص ملفات ZIM.

إذا كنت ستبني المشروع من المصدر، فثبّت .NET 8 SDK.

## فحص الاعتمادات عند التشغيل

يقوم تطبيق سطح المكتب الآن بالتحقق من `zimdump` أثناء بدء التشغيل.

- إذا كان `zimdump` متاحاً، تصبح عملية التحويل جاهزة.
- إذا لم يكن موجوداً، فسيعرض التطبيق تحذيراً ويتيح لك اختيار الملف التنفيذي فوراً.
- يمكن أن يبقى التطبيق مفتوحاً بدون `zimdump`، لكن وظائف التصدير ستظل معطلة حتى يتم ضبطه.

## المزامنة مع WeKnora

أول هدف مدمج لأنظمة RAG هو WeKnora.

يدعم التدفق الحالي في التطبيق:

- عنواناً أساسياً مع مصادقة `API Key` أو `Bearer Token`
- تحميل قواعد المعرفة من خادم WeKnora
- الإنشاء التلقائي بالاسم عند تفعيل ذلك
- إعداد معرّفات نماذج `KnowledgeQA` و `Embedding` و `VLLM` المأخوذة من `/api/v1/models`
- إعادة تطبيق نماذج الدردشة و Embedding ومتعدد الوسائط المهيأة كلما أُنشئت قاعدة معرفة أو بدأت مزامنة
- اختيار مخرجات التحويل المكتملة للمزامنة
- سجل المزامنة، والسجلات لكل مهمة، والتقدم، و ETA، والإيقاف/الاستئناف، ونقاط التوقف القابلة للاستئناف

- `zimdump` يأتي من أدوات Kiwix وليس من هذا المستودع.
